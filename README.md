# Chatbot
# Task 8 - Rule-Based Chatbot

## Objective
Build a simple rule-based chatbot using Python and if-else statements.

## Tools Used
- Python

## Concepts Covered
- Control Flow
- Loops
- Input Handling
- if-elif-else

## How It Works
- The chatbot runs in a loop.
- It takes user input using input().
- It converts input to lowercase using lower().
- It responds based on predefined conditions.
- It exits when the user types "exit".

## How to Run
1. Install Python.
2. Run the file:
   python chatbot.py
